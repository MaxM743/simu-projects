Julia's Set

Historical context

Formal definition

Julia studied rational polynomial expressions such as $f(z)=\sum_{n=0}^Na_nz^n$ but here we will limit the discuss to those of the form : $$f(z)=z^2+c$$ where $z,c\in \mathbb{C}$. The Julia's set denoted $\mathcal{J}$ is represented by all the value $z$ for which the following sequence converges : $$z_{n+1}=z_n^2+c$$

Computation

In order to be able to visualize Julia's set, we have to compute the previous expression to see if it converges. To do so we will assume that if the module of the sequence is greater than 2 then it diverges. Thus we will divide the screen into a grid of coordinates and calculate the sequence to a certain $N$, check when it diverges at each iteration and color the pixel based on this.
